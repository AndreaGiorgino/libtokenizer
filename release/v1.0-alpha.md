## Tokenizer ABI

Name     | Description
:------- | :------------
`get`    | Get the next token
`peek`   | Peek the next token
`ignore` | Ignore the next token/s
`tellg`  | Get the stream offset
`seekg`  | Set the stream offset
`eof`    | Check if the stream has reached eof
`tokens` | Get the tokens in a sequence
